## 九、经验教训

1. **KYC SDK 版本兼容性**：Android 部分老机型无法通过活体，后续需增加降级方案
2. **手机号唯一性**：早期允许一号多客户，后来整改，历史数据清洗耗时 2 周
3. **加密性能**：AES-GCM 早期用 CPU 加密，QPS 高时 CPU 打满，后切换到硬件加密卡
4. **合规提前介入**：产品设计阶段就拉合规同事，避免中期返工
